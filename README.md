# Frontend

React 기반 프론트엔드

## Command

실행시 `vite.config.ts`에서 개발 환경에 맞게 설정을 바꿔아합니다.

```bash
# Install dependencies
$ npm install

# Development
$ npm run dev

# Build
$ npm run build
```
